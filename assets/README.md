# Assets Directory

This directory contains static assets for GH Systems ABC.

## Structure

### Images (`assets/images/`)
- **Risk score breakdowns** - Visualizations of risk assessments
- **System diagrams** - Architecture and ontology diagrams
- **Logos** - GH Systems branding assets

### Visualizations (`assets/visualizations/`)
- **Python scripts** - Scripts for generating visualizations
- **Generated charts** - Output from visualization scripts

## Usage

Visualization scripts can be run to generate charts and diagrams:

```bash
python3 assets/visualizations/generate_doge_government_risk_visual.py
python3 assets/visualizations/generate_cia_risk_visual.py
```
